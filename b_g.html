USE:-Background-image tag is use to add image to the background of our page.
For Giving Background-image To our page You Need to Use Background-image tag in <style>

How to use:-
<HTML>
<head>
<style>
h1{
Background-image:url(image.jpg);
}
</style>
</head>
<body>
<h1>hello guys...</h1>
</body>
</HTML>
Notice:-
Always use url before adding image to our page
