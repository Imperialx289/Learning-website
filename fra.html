Hello guys! in this part we are going to see how to add Frameset to our web page
Tag which use:-
<frameset> as opening and for closing</frameset>

How to use it:-

<html>
<head>
</head>

<frameset rows="20%,80%">

<frame src=top.html>

<frameset cols="15%,85%">

<frame src=left.html>

<frame src=right.html>

</frameset>

</html>

NOTICE:-
Always save HTML file with extention .HTML
