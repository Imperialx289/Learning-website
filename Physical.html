Physical tags

MEANING:-
Tag which come in <body> this Tag is know as physical Tag.



BOLD : It is used to bold the text.
Syntax : <B>TEXT </B>
eg : Text



UNDERLINE : It is used to underline particular text.
Syntax : <U>TEXT </U>
eg : text



ITALIC : It is use to make text italic.
Syntax : <I>TEXT </I>
eg : text



SUBSCRIPT : It is used to subscript the text .
Syntax : <SUB>TEXT </SUB>
eg:H2o



SUPERSCRIPT : It is used to supscript the text .
Syntax : <SUP>TEXT </SUP>
eg :22



BIG : It is used to increase size of the text .
Syntax : <BIG>TEXT </BIG>
eg :Text



Small : It is used to Decrease size of the text .
Syntax : <Small>TEXT </Small>
eg :Text



Strikethrough : It is used to strikethrough the text .
Syntax : <Strikethtrough>TEXT </Strikethtrough>
eg :Text
