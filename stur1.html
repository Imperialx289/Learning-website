Sturture Tags:-
web page have to basic containers know as <head> and <body>.

1.The <head> tag which contain information about web page such as title for the page javascript style (css) meta information etc.Whatever is inside the head tag does not display on the web page itself

2.The <body> tag containing contain that actually display in a browser for e.g

<HTML>
<head>
< title>
The title of your web page goes here
</title>
</head>
<body>
The real contain show on your web page
</body>
</HTML>
