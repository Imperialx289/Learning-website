Hello guys! in this part we are going to see how to create Table to our web page
Tag which use:-
<table> as opening and for closing</table>

How to use it:-

<html>
<head>
</head>
<body>
<table border="1" width="200" height="200">

<tr>

<td>1 </td>
<td>2 </td>

</tr>

<tr>

<td>3 </td>
<td>4 </td>

</tr>

</table>
</body>
</html>

NOTICE:-
Always save HTML file with extention .HTML
<tr> is stand for table rows.It help us to create rows is table
<td> is stand for table data which help us to insert data in rows of table.

OUTPUT:-

1 2
3 4
