USE:-Font-variant tag is use to change letters of text to our html page
For giving Font-variantTo our page we Need to Use Font-variant tag in <style>

How to use:-
<HTML>
<head>
<style>
h1{
Font-variant:small-caps;
}
</style>
</head>
<body>
<h1>hello guys...</h1>
</body>
</HTML>



output:-
HELLO GUYS...
