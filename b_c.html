USE:-Background-color tag is use to give color to the background of our page.
For Giving Background-color To our page You Need to Use Background-color tag in <style>

How to use:-
<HTML>
<head>
<style>
h1{
Background-color:Blue;
}
</style>
</head>
<body>
<h1>hello guys...</h1>
</body>
</HTML>



output:-
hello guys...
