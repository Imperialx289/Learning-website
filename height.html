USE:-Height tag is use to give Height to our elemant in our html page
For Giving Height To our page we Need to Use Height tag in <style>

How to use:-
<HTML>
<head>
<style>
h1{
Height:28px;
border:solid green;
}
</style>
</head>
<body>
<h1>hello guys...</h1>
</body>
</HTML>



output:-
hello guys...
